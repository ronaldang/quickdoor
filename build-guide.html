Part 1: Create New Shortcut

  1. 1.

    Open the Shortcuts app on your iPhone

    It's the colorful square icon, usually in your utilities folder

  2. 2.

    Tap the + button in the top right corner

  3. 3.

    Name your shortcut "QuickDoor"

    Tap "Shortcut Name" at the top to rename

Part 2: Add Actions (In Order)

Action 1: Add Opening Message

  • Tap "Add Action"
  • Search for "Text"
  • Add the Text action
  • Type: Hello! Here's my delivery address:

Action 2: Get Current Location

  • Tap the + below your first action
  • Search for "Get Current Location"
  • Add this action (no settings needed)

Action 3: Get Maps URL

  • Tap +
  • Search for "Get Maps URL from"
  • Add this action
  • It should automatically use "Current Location"

Action 4: Add Custom Details

  • Tap +
  • Add another "Text" action
  • Type your details (customize these!):
Floor: 3
Apartment: 301
Building: White building with green door
Near: Starbucks on the corner

Action 5: Select Photos

  • Tap +
  • Search for "Select Photos"
  • Add this action
  • Turn ON "Select Multiple"

‚ö†Ô∏è You'll select actual photos when you run it

Action 6: Combine Everything

  • Tap +
  • Search for "Combine"
  • Add "Combine Text" action
  • Set to combine with "New Lines"

Action 7: Copy to Clipboard

  • Tap +
  • Search for "Copy to Clipboard"
  • Add this action
  • Input should be "Combined Text"

Action 8: Open WhatsApp

  • Tap +
  • Search for "Open App"
  • Add this action
  • Choose "WhatsApp"

üì± The message will be in your clipboard to paste

Part 3: Test Your Shortcut

  1. 1.

    Tap "Done" to save your shortcut

  2. 2.

    Tap the shortcut to run it

    It will ask for location permission (allow)

  3. 3.

    Select 2-3 photos of your building when prompted

  4. 4.

    WhatsApp will open - paste the message and send!

‚úÖ Success! Your shortcut is ready to use

Optional: Add to Home Screen

For even faster access:

  1. In Shortcuts app, long-press QuickDoor
  2. Tap "Share"
  3. Select "Add to Home Screen"
  4. Choose an icon and name
  5. Tap "Add"

Share With Friends

Once you've built it, you can share your shortcut:

  1. Long-press your QuickDoor shortcut
  2. Tap "Share"
  3. Select "Copy iCloud Link"
  4. Share the link with friends!

Note: Recipients need to have "Allow Untrusted Shortcuts" enabled in Settings ‚Üí Shortcuts
