Prepare Your Building Photos

Before activating the shortcut, prepare photos to send to delivery drivers:

  1. Open the Photos app
  2. Select 2-3 clear photos of your building:
    • Building exterior view
    • Main entrance photo
    • Nearby landmark (e.g., famous store)
  3. Tap Share and select "Add to Album"
  4. Create a new album named Delivery Photos

üí° Tip: You only need to set up these photos once and they'll be used automatically

3

Configure Your Details

Now let's customize the shortcut with your information:

  1. Open the Shortcuts app
  2. Tap on the QuickDoor shortcut
  3. Tap the three dots (...) to edit
  4. Find the text fields marked "Enter your info" and update:
    • Floor number
    • Apartment number
    • Additional instructions (e.g., "Near the security gate")
  5. Tap Done at the top
4

Enable Automation

Final step - link the shortcut to incoming calls:

  1. In Shortcuts app, select Automation tab at the bottom
  2. Tap + to add a new automation
  3. Select "Create Personal Automation"
  4. Choose "Phone Call"
  5. Configure:
    • From: Anyone
    • ‚úì Has caller ID (important!)
    • ‚úó Is in my contacts (unchecked)
  6. Tap Next
  7. Add Action ‚Üí Run Shortcut ‚Üí select QuickDoor
  8. Turn off "Ask Before Running" for automatic operation
  9. Tap Done

Usage Tips

Delivery Mode

Enable the shortcut only when expecting deliveries. Disable it during normal times from Shortcuts ‚Üí Automation

Test First

Ask a friend to call from an unsaved number to test the shortcut

Location Updates

Location is captured automatically when the call comes. Make sure GPS is enabled

Troubleshooting Common Issues

Shortcut doesn't run automatically

Make sure "Ask Before Running" is turned off in the automation settings

WhatsApp doesn't open

Ensure WhatsApp is installed and you're logged in

Photos don't appear

Make sure you created an album named exactly "Delivery Photos"

Location error message

Go to Settings ‚Üí Privacy ‚Üí Location Services ‚Üí Shortcuts ‚Üí Always

Shortcut runs for all calls

Make sure "Is in my contacts" is unchecked in the automation trigger
